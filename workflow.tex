\subsection{Workflow}
The usual git workflow looks like:
\begin{enumerate}
  \item{Do some programming.}
  \item{\cmd{git status} to see what files I changed.}
  \item{\cmd{git diff [file]} to see exactly what I modified.}
  \item{\cmd{git commit -a -m [message]} to commit.}
\end{enumerate}
